MyPerfectice Genie

Recomended Browser : Microsoft Edge

  • Install This Extension and Enable Copy on Myperfectice website
  • Start Exam.
  • Press 'ESC' key
  • Inspect Tab!
  • Open MyPerfectice Genie Panel.
  • Click Start!
  • Refresh Page
  • Press CTRL + V to paste the code in editor
  • NOTE : After finishing test if your marks is not updated then click on 'resume' button just below 'Take Assessment' button and complete the test again.
